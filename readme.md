# Imooc

采用vuecli以及springcloud alibaba开发前后端分离的在线视频课程系统,本项目使用Nacos作为服务发现组件，有登录、展示课程、课程详情、在线视频播放、资源下载、文件管理、自定义权限以及拦截、后台CMS管理等功能
